Aplicación de Gestión de Hotel

Esta aplicación permite gestionar las reservas y clientes de un hotel. Con ella, puedes realizar operaciones como:

  • Agregar, editar y eliminar clientes.
  • Realizar, editar y eliminar reservas.
  • Consultar la disponibilidad de habitaciones.
  • Ver estadísticas y generar reportes.

La aplicación está diseñada para ser intuitiva, permitiendo una experiencia de usuario sencilla y eficiente.
